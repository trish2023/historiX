# HistoriX - Journey Through Time

An interactive web application for exploring the rich tapestry of human history, with a special focus on Indian history.

## Features

- **Interactive Map** - Global historical atlas with markers for significant historical sites
- **Historical Timeline** - Slider-based timeline covering Indian history from the Paleolithic Period to modern day
- **History Quiz** - Test your knowledge of Indian history with an interactive quiz
- **Community** - Share and discuss historical discoveries with other history enthusiasts
- **Music Library** - Curated collection of Indian classical, folk, and fusion music via Spotify embeds
- **AI Insights** - AI-powered historical assistant for deeper exploration

## Getting Started

1. Clone or download this repository
2. Open `withenewtimelinept2.html` in a web browser
3. Navigate using the header links to explore different sections
